# PrepFor20LPA

Practice repository for building strong Java fundamentals on the path to 20 LPA+: conditionals, loops, and classic beginner problems. The code is organized as a simple IntelliJ IDEA project with a `BasicsOfJava` module and plain `javac/java` compatibility.  
> Latest update: *Day 1 + Day 2 Codes and Learnings* (see commit history).

---

## Prerequisites

- **JDK 17** (the project uses a Java 17 SDK in the IDE config; e.g., Corretto 17)
- Any Java-friendly editor/IDE (IntelliJ IDEA recommended)

---

## Getting Started

### Option A: Run with IntelliJ IDEA
1. Open **File → Open…** and select the repository folder.
2. Ensure the project SDK is **JDK 17**.
3. Right-click a file (e.g., `Main.java` or any class with `public static void main`) → **Run**.

### Option B: Run from the Terminal
```bash
# from the repository root
cd BasicsOfJava/src

# compile a single class (example)
javac BasicProblems/CalculatorProgram.java

# run it (use the fully qualified class name if a package is declared)
java BasicProblems.CalculatorProgram
```
